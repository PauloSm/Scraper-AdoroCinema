# Web Scrapper do Site AdoroCinema

# Pré-requisitos
* Python 3.x<br>
* Requests<br>
* Selenium<br>
* BeautifulSoup4

# Execução do código
python movieScraper.py nome do filme

# Resultado
Se o filme for encontrado as seguintes informações serão retornadas:<br>
* Ano de produção;<br>
* Tipo de filme;<br>
* Orçamento;<br>
* Idiomas;<br>
* Pontuação;<br>
